Класс Program является точкой входа в приложение.
Он содержит метод Main, который выполняется при запуске.
В данном приложении реализована многопоточная обработка файлов.
Используются следующие технологии:
- async/await для асинхронных операций
- Mutex для синхронизации потоков
- Структуры данных для хранения результатов
